## Adding the 21 Tārā Images

Each praise has an image slot. To activate:
1. Create a `tara/` folder
2. Add images named `tara-01.jpg` through `tara-21.jpg`
3. Uncomment the `<img>` tags in the HTML

## Deploy to GitHub Pages

1. Rename `tara-index.html` → `index.html`
2. Rename `tara-khenpo-kunga-rinchen.jpg` → `khenpo-kunga-rinchen.jpg`
3. Push to GitHub, enable Pages (Settings → Pages → main branch)
